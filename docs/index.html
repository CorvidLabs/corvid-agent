What is CorvidAgent?

CorvidAgent is an open-source autonomous AI agent platform that combines Claude’s reasoning with Algorand’s on-chain identity. Deploy self-improving agents that can write code, coordinate in councils, execute scheduled tasks, and communicate through encrypted on-chain messaging — all with a verifiable cryptographic identity.

Features

🧠

Agent Orchestration

Multi-agent council system with configurable personas. Agents deliberate, vote, and reach consensus. Supports both Claude and local Ollama models.

💬

AlgoChat Messaging (optional)

On-chain encrypted messaging via Algorand. Agents communicate with cryptographic identity and verifiable message history. Works fully without it — chat via the web UI + SQLite persistence.

⏰

Scheduler

Cron and interval-based autonomous task execution. Agents run on schedule with approval policies (auto, owner, council) and budget limits.

🧊

Ollama / Local Models

Run Qwen, LLama, Mistral, and other local models. Weight-based concurrency prevents memory thrashing. Real-time tok/s metrics.

🛠

Work Tasks

Spawn autonomous agent sessions on dedicated branches. Agents implement changes, run validation, and open PRs.

📱

Dashboard & Mobile

Angular UI with analytics, system logs, work tasks, settings, and QR-based mobile connect. Dark cyberpunk aesthetic.

🌐

Graph Workflows

Define multi-step workflows as DAGs. Condition nodes, parallel fork/join, delays, and suspend/resume across sessions.

🤝

A2A Protocol

Agent-to-Agent interoperability via Google A2A. Agents publish Agent Cards and discover remote agent capabilities.

🔍

Observability

OpenTelemetry tracing, Prometheus metrics, and immutable audit logging. Full trace context across agent sessions.

📢

Multi-Channel Comms

Notifications and two-way questions via Discord, Telegram, GitHub Issues, and AlgoChat. Per-agent config with delivery tracking.

🌳

AST Parser

Tree-sitter AST analysis for smarter code understanding. Language-aware symbol extraction across work tasks.

💬

Owner Communication

Two-way agent-to-owner questions with blocking polls. Responses via GitHub checkboxes, Telegram keyboards, or AlgoChat.

Quick Start

terminal 
# Clone & install
$ git clone https://github.com/CorvidLabs/corvid-agent.git
$ cd corvid-agent
$ bun install

# Configure
$ cp .env.example .env
$ # Add your ANTHROPIC_API_KEY and ALGO_MNEMONIC

# Run
$ bun dev          # API server on :3000
$ bun dev:client   # Angular UI on :4200

Security & Trust

Wallet Auth

Every agent and user authenticates via Algorand wallet. No passwords, no cookies — just cryptographic keys.

Encrypted Memory

Agent memories are encrypted and stored on-chain. Only the owning agent can decrypt and recall its own notes.

Credit System

ALGO-backed credits prevent abuse. Each conversation turn consumes credits purchased with native tokens.

Audit Logging

Immutable, insert-only audit log with trace IDs. Full forensic trail of every agent action for compliance.

How it Works

01

Deploy

Clone the repo, add your API key, and run bun dev. Your agent boots with a unique Algorand wallet identity.

02

Configure

Set up schedules, workflows, and council personas via the Dashboard UI or REST API. Define what your agent does autonomously.

03

Orchestrate

Agents execute work tasks on branches, deliberate in councils, communicate on-chain, and self-improve — with full audit trails.
